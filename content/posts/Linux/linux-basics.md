## Linux 简介

Linux 是一个开源的类 Unix 操作系统，具有强大、稳定、安全等特点。

## 基本概念

- **发行版**：Ubuntu、CentOS、Debian 等
- **内核**：系统核心
- **Shell**：命令行界面

## 常用命令

```bash
# 查看目录
ls -la

# 切换目录
cd /path/to/directory

# 创建文件
touch filename.txt
```
